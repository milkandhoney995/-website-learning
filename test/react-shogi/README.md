# Shogi App

A Japanese Shogi (chess) game built with React.js and TypeScript.

## Features

- 9x9 Shogi board
- Piece movement for basic pieces
- Turn-based gameplay
- Move history

## Getting Started

In the project directory, run:

### `npm start`

Runs the app in development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `npm run build`

Builds the app for production.

## How to Play

Click on a piece to select it, then click on a valid square to move. Only basic movement rules are implemented.
